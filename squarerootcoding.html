Create a DYI Square Root function in Python

You've learnt to take the square root of any number in Python with this: 


import math
print(math.sqrt(9)) #3.0

Or this:


print(9 ** (1/2)) #3.0

But where's the fun in that? When you can make your own DIY sqrt(x) function.

Let's look at how you can approxomate square roots before calculators:

\[ \sqrt{s} \approx{} \sqrt{a} + \frac{s - a}{2\sqrt{a}} \]

Where a is the closest perfect square to s. The more interations of this you do, the more accurate the result gets.

Let's do an example with the square root of 37 over 2 iterations of this

Iteration 1

\[ \sqrt{37} \approx{} \sqrt{36} + \frac{37 - 36}{2\sqrt{36}} \approx{} 6.083 \]

Iteration 2

\[ \sqrt{a} = 6.083 \therefore{} a = 37.002889 \]
\[ \sqrt{37} \approx{} 6.083 + \frac{37 - 37.002889}{2 \cdot{} 6.083} \approx{} 6.08276 \]
\[ \sqrt{37} = 6.08276... \]

See how accurate it is to the accual answer? With Python, with more iterations of this, the more time it takes to calculate this, but also the more accurate.
